Personal Assistant

This is a simple assistant built using OpenAI gpt-4o-mini model. It is a single page app with JavaScript frontend that calls API endpoints in the Python Flask backend. It uses Flask sessions to maintain conversation history, and has a function to clean up stale sessions.

Coming Soon: I am working on a specialized assistant. That model needs additional tuning so that the embeddings will enable the retrieval of the most relevant information from ChromaDB.

Log File Processor

In my current role, I've been hands on in cloud migration work. I've added these tools here to share with my teammates. The Log File Processor tool works for a specific log file that doesn't contain any confidential data.

The List Comparison tool is used to compare a primary or source file that lists unique identifiers with a secondary or destination file listing unique identifiers. I made a few improvements to reduce the processing time for very large files.

Umbrella App

Heading out? Wondering if you should bring an umbrella? This app will keep you dry.

I used Virtual crossing weather API to retrieve the chance of rain, and Twilio API to send the SMS message that lets you know whether you need an umbrella. The background image is by ChatGPT 4o. I struggled a bit using tailwindcss and plan to come back and improve the look of the site.

This Python Flask app is an adaptation of the rain tracker project in Angela Yu's 100 Days of Code: The Complete Python Pro Bootcamp.

Dad Jokes
